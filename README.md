# 🔐 Ransomware Review – Critical Infrastructure (Post-2017)

## 📄 Overview
This repository contains my Secure Software Systems assignment, which presents a **systematic literature review** on ransomware evolution and defense mechanisms in critical infrastructure sectors.

The study focuses on modern ransomware trends such as:
- Double and Triple Extortion
- Ransomware-as-a-Service (RaaS)
- Targeted attacks on critical sectors (Healthcare, Energy, Transport)

## 🎯 Objective
The objective of this research is to:
- Analyze ransomware techniques developed after 2017
- Evaluate current defense mechanisms
- Identify research gaps in cybersecurity protection

## 🧠 Key Topics Covered
- Evolution of ransomware (Pre & Post-2017)
- Advanced attack techniques
- RaaS business model
- Sector-specific impacts
- Defense strategies:
  - Behavioral Detection
  - AI/ML Security
  - Zero Trust Architecture
  - MITRE ATT&CK & NIST Frameworks

## 📊 Methodology
- Followed **PRISMA framework**
- Reviewed **25 peer-reviewed research papers**
- Sources include:
  - IEEE Xplore
  - SpringerLink
  - ScienceDirect
  - ACM Digital Library
